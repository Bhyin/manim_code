# manim code
#### 用来保存一些manim项目的代码。

---

manim是3blue1brown开发的用于制作数学动画的python库。manim在github上开源。

作者3B1B：<https://github.com/3b1b>

我惯用的manim版本是manimGL，但由于一些无法解决的问题而暂时改用manimCE，等到manimGL的1.7.0版本release之后会继续使用manimGL。
